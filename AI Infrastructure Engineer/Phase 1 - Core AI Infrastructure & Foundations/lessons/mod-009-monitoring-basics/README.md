# Module 009: Monitoring and Observability Basics

Welcome to Module 009, where you'll learn the fundamentals of monitoring, logging, metrics collection, visualization, alerting, and observability. This module covers setting up Prometheus for collecting time-series metrics, Grafana for interactive dashboards, structured logging pipelines, and Alertmanager for incident response. You will also learn how to monitor machine learning serving pipelines and workflow orchestrators to ensure production system reliability.
