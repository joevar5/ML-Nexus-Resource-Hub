# Module 004: Machine Learning Basics for Infrastructure Engineers

This module bridges the gap between traditional infrastructure engineering and the specialized world of AI/ML systems. You'll learn exactly what infrastructure engineers need to know about ML to effectively build, deploy, and maintain ML systems. You'll explore PyTorch and TensorFlow basics, resource requirements (CPU, GPU, memory), model formats like ONNX, and model serving workflows.
