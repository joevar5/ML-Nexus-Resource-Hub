# Module 11: Multi-GPU & Distributed Training Foundations

Welcome to Module 11! This module covers the theoretical and practical foundations of scaling deep learning model training across multiple GPUs and nodes.

## Objectives
- Understand Data Parallelism (DP) and Distributed Data Parallelism (DDP).
- Set up PyTorch DDP pipelines.
- Trace memory leaks and optimize GPU memory utilization using mixed precision (AMP) and activation checkpointing.
